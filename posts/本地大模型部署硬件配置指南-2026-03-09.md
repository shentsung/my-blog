# 本地大模型部署硬件配置指南

> 分析日期：2026-03-09

---

## 核心原则

> **显存 > 内存 > CPU**

大模型部署的核心瓶颈是**显存容量**，优先保证显存，再考虑其他组件。

---

## 显存与模型参数对照表

| 显存 | 可运行模型（量化） | 适用场景 |
|------|-------------------|----------|
| 8GB | 7B (~4bit) | 入门体验 |
| 12GB | 7B (~4bit) / 8B (~8bit) | 日常开发 |
| 16GB | 13B (~4bit) / 14B (~8bit) | 专业开发 |
| 24GB | 34B (~4bit) / 70B (~4bit) | 高端推理 |
| 48GB+ | 70B+ (~8bit) / 405B (~4bit) | 满血运行 |

## 特殊方案：纯 CPU 推理

如果完全没有预算买显卡，可以考虑**纯 CPU 方案**：

| 价格 | 配置 | 可运行模型 |
|------|------|------------|
| ¥1,500 | 64GB 内存 + 洋垃圾服务器 | 7B 模型 |
| ¥2,500 | 128GB 内存 | 13B 模型 |

**缺点：** 推理速度极慢（~1-3 tok/s），但成本极低。

## 选购建议

### ✅ 推荐品牌

- **GPU**：优先 RTX 4090 / 4080 SUPER / 4060 Ti 16GB
- **CPU**：Intel i5/i7 或 AMD R5/R7 足够
- **内存**：32GB 起步，64GB 最佳

### ❌ 避坑指南

1. **不要买 8GB 显存显卡** - 无法运行 14B+ 模型
2. **RTX 4090 假货多** - 建议京东/天猫正规渠道
3. **电源要留余量** - 建议 650W 以上
4. **注意散热** - 大模型推理发热量大
